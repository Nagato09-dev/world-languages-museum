# OX

**Ano de Criação:** 1996

## Descrição

Esta pasta contém exemplos de código em OX, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.ox** - Olá Mundo
2. **exemplo_2.ox** - Variáveis e Tipos
3. **exemplo_3.ox** - Controle de Fluxo
4. **exemplo_4.ox** - Loops (Repetições)
5. **exemplo_5.ox** - Funções
6. **exemplo_6.ox** - Estruturas de Dados
7. **exemplo_7.ox** - Tratamento de Erros
8. **exemplo_8.ox** - Manipulação de Strings
9. **exemplo_9.ox** - Matemática
10. **exemplo_10.ox** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de OX.
Cada arquivo contém comentários explicativos em português.
