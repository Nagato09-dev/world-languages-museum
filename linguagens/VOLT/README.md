# VOLT

**Ano de Criação:** 2012

## Descrição

Esta pasta contém exemplos de código em VOLT, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.volt** - Olá Mundo
2. **exemplo_2.volt** - Variáveis e Tipos
3. **exemplo_3.volt** - Controle de Fluxo
4. **exemplo_4.volt** - Loops (Repetições)
5. **exemplo_5.volt** - Funções
6. **exemplo_6.volt** - Estruturas de Dados
7. **exemplo_7.volt** - Tratamento de Erros
8. **exemplo_8.volt** - Manipulação de Strings
9. **exemplo_9.volt** - Matemática
10. **exemplo_10.volt** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de VOLT.
