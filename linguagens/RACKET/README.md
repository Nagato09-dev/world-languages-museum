# RACKET

**Ano de Criação:** 1995

## Descrição

Esta pasta contém exemplos de código em RACKET, organizados de forma progressiva para facilitar o aprendizado.

2. **exemplo_2.racket** - Variáveis e Tipos

4. **exemplo_4.racket** - Loops (Repetições)
5. **exemplo_5.racket** - Funções
6. **exemplo_6.racket** - Estruturas de Dados
7. **exemplo_7.racket** - Tratamento de Erros
8. **exemplo_8.racket** - Manipulação de Strings
9. **exemplo_9.racket** - Matemática
10. **exemplo_10.racket** - Conceito Avançado / Algoritmos
