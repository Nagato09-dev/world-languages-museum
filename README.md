# 🌍 Museu de Linguagens do Mundo

Bem-vindo ao **Museu de Linguagens do Mundo**, o maior acervo digital e interativo de tecnologias de programação do GitHub! Este projeto é uma celebração da evolução do software, contendo informações, exemplos e exercícios para mais de **500 linguagens diferentes**.

---

## 🚀 Experiência Imersiva (GitHub Pages)
O projeto conta com uma interface visual **completamente personalizada**, com estética de museu digital futurista.

👉 **(https://Nagato09-dev.github.io/museu-de-linguagens-do-mundo/)**

---

## ✨ Novas Funcionalidades (v2.0)

A versão mais recente trouxe uma reformulação profunda na experiência do usuário:

*   🎨 **Identidade Visual Futurista:** Tema "Dark Mode" profundo com paleta Dourada e Ciano, tipografia Orbitron e fundo de partículas animado em tempo real.
*   ⭐ **Sistema de Favoritos:** Marque suas linguagens preferidas e acesse-as rapidamente em uma categoria dedicada. Os dados persistem no seu navegador.
*   🔍 **Busca e Filtros Inteligentes:** Pesquisa em tempo real e menu de categorias com ícones e contadores dinâmicos.
*   🖥️ **Syntax Highlighting:** Visualização de código profissional com destaque de sintaxe em todos os exemplos e exercícios.
*   📋 **Cópia Rápida:** Botão dedicado para copiar blocos de código com um clique.
*   🎲 **Linguagem Aleatória:** Explore o acervo de forma lúdica com o seletor de tecnologia surpresa.
*   📊 **Painel de Estatísticas:** Visualize a distribuição das linguagens por categoria e o seu progresso de exploração.
*   📱 **Totalmente Responsivo:** Interface otimizada para desktops, tablets e smartphones.

## 📂 Estrutura do Acervo
O museu está organizado em pastas individuais dentro de `linguagens/`. Cada tecnologia possui:

1.  **README.md:** História, criador, ano de lançamento e curiosidades.
2.  **Exemplos:** Arquivos de código demonstrando a sintaxe básica e avançada.
3.  **Exercícios:** Desafios práticos para testar seus conhecimentos na linguagem.

## 🛠️ Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript (Vanilla)
- **Estilo:** Google Fonts (Orbitron & Roboto), Font Awesome 6
- **Funcionalidades:** Canvas API (Fundo animado), LocalStorage (Favoritos & Histórico)
- **Bibliotecas:** [Highlight.js](https://highlightjs.org/) para Syntax Highlighting
