# LLVM

**Ano de Criação:** 2003

## Descrição

Esta pasta contém exemplos de código em LLVM, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.llvm** - Olá Mundo
2. **exemplo_2.llvm** - Variáveis e Tipos
3. **exemplo_3.llvm** - Controle de Fluxo
4. **exemplo_4.llvm** - Loops (Repetições)
5. **exemplo_5.llvm** - Funções
6. **exemplo_6.llvm** - Estruturas de Dados
7. **exemplo_7.llvm** - Tratamento de Erros
8. **exemplo_8.llvm** - Manipulação de Strings
9. **exemplo_9.llvm** - Matemática
10. **exemplo_10.llvm** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de LLVM.
