# AMPL

**Ano de Criação:** 1990

## Descrição

Esta pasta contém exemplos de código em AMPL, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.ampl** - Olá Mundo
2. **exemplo_2.ampl** - Variáveis e Tipos
3. **exemplo_3.ampl** - Controle de Fluxo
4. **exemplo_4.ampl** - Loops (Repetições)
5. **exemplo_5.ampl** - Funções
6. **exemplo_6.ampl** - Estruturas de Dados
7. **exemplo_7.ampl** - Tratamento de Erros
8. **exemplo_8.ampl** - Manipulação de Strings
9. **exemplo_9.ampl** - Matemática
10. **exemplo_10.ampl** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de AMPL.
Cada arquivo contém comentários explicativos em português.
