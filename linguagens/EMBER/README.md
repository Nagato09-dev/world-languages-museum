# EMBER

## Descrição

Esta pasta contém exemplos de código em EMBER, organizados de forma progressiva para facilitar o aprendizado.

2. **exemplo_2.ember** - Variáveis e Tipos
3. **exemplo_3.ember** - Controle de Fluxo
4. **exemplo_4.ember** - Loops (Repetições)
5. **exemplo_5.ember** - Funções
6. **exemplo_6.ember** - Estruturas de Dados
7. **exemplo_7.ember** - Tratamento de Erros
8. **exemplo_8.ember** - Manipulação de Strings
9. **exemplo_9.ember** - Matemática
10. **exemplo_10.ember** - Conceito Avançado / Algoritmos
