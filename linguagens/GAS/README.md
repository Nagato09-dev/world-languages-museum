## Descrição

Esta pasta contém exemplos de código em GAS, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.gas** - Olá Mundo
2. **exemplo_2.gas** - Variáveis e Tipos
3. **exemplo_3.gas** - Controle de Fluxo
4. **exemplo_4.gas** - Loops (Repetições)
5. **exemplo_5.gas** - Funções
6. **exemplo_6.gas** - Estruturas de Dados
7. **exemplo_7.gas** - Tratamento de Erros
8. **exemplo_8.gas** - Manipulação de Strings
9. **exemplo_9.gas** - Matemática
10. **exemplo_10.gas** - Conceito Avançado / Algoritmos
