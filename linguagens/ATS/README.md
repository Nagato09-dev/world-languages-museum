# ATS

**Ano de Criação:** 2003

## Descrição

Esta pasta contém exemplos de código em ATS, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.ats** - Olá Mundo
2. **exemplo_2.ats** - Variáveis e Tipos
3. **exemplo_3.ats** - Controle de Fluxo
4. **exemplo_4.ats** - Loops (Repetições)
5. **exemplo_5.ats** - Funções
6. **exemplo_6.ats** - Estruturas de Dados
7. **exemplo_7.ats** - Tratamento de Erros
8. **exemplo_8.ats** - Manipulação de Strings
9. **exemplo_9.ats** - Matemática
10. **exemplo_10.ats** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de ATS.
Cada arquivo contém comentários explicativos em português.
