Esta pasta contém exemplos de código em OPENCL, organizados de forma progressiva para facilitar o aprendizado.

2. **exemplo_2.opencl** - Variáveis e Tipos

4. **exemplo_4.opencl** - Loops (Repetições)

10. **exemplo_10.opencl** - Conceito Avançado / Algoritmos
