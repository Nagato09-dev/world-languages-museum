# MOONSCRIPT

**Ano de Criação:** 2011

## Descrição

Esta pasta contém exemplos de código em MOONSCRIPT, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

2. **exemplo_2.moonscript** - Variáveis e Tipos
3. **exemplo_3.moonscript** - Controle de Fluxo
4. **exemplo_4.moonscript** - Loops (Repetições)
5. **exemplo_5.moonscript** - Funções
6. **exemplo_6.moonscript** - Estruturas de Dados
7. **exemplo_7.moonscript** - Tratamento de Erros
8. **exemplo_8.moonscript** - Manipulação de Strings
9. **exemplo_9.moonscript** - Matemática
10. **exemplo_10.moonscript** - Conceito Avançado / Algoritmos
