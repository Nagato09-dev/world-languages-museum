**Ano de Criação:** 2003

## Descrição

Esta pasta contém exemplos de código em GLSL, organizados de forma progressiva para facilitar o aprendizado.

2. **exemplo_2.glsl** - Variáveis e Tipos

4. **exemplo_4.glsl** - Loops (Repetições)
5. **exemplo_5.glsl** - Funções
6. **exemplo_6.glsl** - Estruturas de Dados
7. **exemplo_7.glsl** - Tratamento de Erros
8. **exemplo_8.glsl** - Manipulação de Strings
9. **exemplo_9.glsl** - Matemática
10. **exemplo_10.glsl** - Conceito Avançado / Algoritmos
