# GNUPLOT

**Ano de Criação:** 1986

## Descrição

Esta pasta contém exemplos de código em GNUPLOT, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.gnuplot** - Olá Mundo
2. **exemplo_2.gnuplot** - Variáveis e Tipos
3. **exemplo_3.gnuplot** - Controle de Fluxo
4. **exemplo_4.gnuplot** - Loops (Repetições)
5. **exemplo_5.gnuplot** - Funções
6. **exemplo_6.gnuplot** - Estruturas de Dados
7. **exemplo_7.gnuplot** - Tratamento de Erros
8. **exemplo_8.gnuplot** - Manipulação de Strings
9. **exemplo_9.gnuplot** - Matemática
10. **exemplo_10.gnuplot** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de GNUPLOT.
Cada arquivo contém comentários explicativos em português.
