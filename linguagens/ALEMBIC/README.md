# ALEMBIC

**Ano de Criação:** 2006

## Descrição

Alembic é uma ferramenta de migração de banco de dados leve para SQLAlchemy, escrita em Python. Ele fornece um sistema de controle de revisão para esquemas de banco de dados, permitindo que os desenvolvedores gerenciem e apliquem alterações incrementais ao esquema de um banco de dados ao longo do tempo. Os exemplos abaixo demonstram como usar o Alembic para tarefas comuns de migração.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos, adaptados para o contexto de migrações de banco de dados com Alembic em Python:

1. **exemplo_1.py** - Estrutura básica de uma migração (equivalente a "Olá Mundo" no contexto de migração)
2. **exemplo_2.py** - Adição de uma coluna a uma tabela
3. **exemplo_3.py** - Remoção de uma coluna de uma tabela
4. **exemplo_4.py** - Alteração do tipo de uma coluna
5. **exemplo_5.py** - Criação de uma nova tabela
6. **exemplo_6.py** - Exclusão de uma tabela
7. **exemplo_7.py** - Renomear uma tabela
8. **exemplo_8.py** - Renomear uma coluna
9. **exemplo_9.py** - Inserção de dados iniciais (seed data)
10. **exemplo_10.py** - Operações de banco de dados personalizadas
