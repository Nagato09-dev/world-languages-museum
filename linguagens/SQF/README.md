# SQF

**Ano de Criação:** 2002

## Descrição

Esta pasta contém exemplos de código em SQF, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.sqf** - Olá Mundo
2. **exemplo_2.sqf** - Variáveis e Tipos
3. **exemplo_3.sqf** - Controle de Fluxo
4. **exemplo_4.sqf** - Loops (Repetições)
5. **exemplo_5.sqf** - Funções
6. **exemplo_6.sqf** - Estruturas de Dados
7. **exemplo_7.sqf** - Tratamento de Erros
8. **exemplo_8.sqf** - Manipulação de Strings
9. **exemplo_9.sqf** - Matemática
10. **exemplo_10.sqf** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de SQF.
