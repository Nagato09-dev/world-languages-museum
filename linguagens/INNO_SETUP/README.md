# Inno_setup

A linguagem INNO_SETUP possui raízes no desenvolvimento de software especializado e evoluiu para atender necessidades específicas de sua comunidade.

Atualmente, é utilizada em nichos acadêmicos, sistemas legados ou projetos de código aberto que valorizam suas características únicas.

## ✨ Curiosidade
