# ZAP (Zap Language - gjmcn/zap)

**Ano de Criação:** 2013

## Descrição

Zap é uma linguagem de programação que compila para JavaScript, conhecida por sua sintaxe limpa e concisa, e operadores de alto nível. Ela permite a interoperabilidade com objetos e bibliotecas JavaScript. Os exemplos abaixo demonstram conceitos básicos adaptados para a sintaxe provável de Zap, focando na sua natureza de alto nível e concisa.
