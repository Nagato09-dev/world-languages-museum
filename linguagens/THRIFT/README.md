# THRIFT

**Ano de Criação:** 2007

## Descrição

Esta pasta contém exemplos de código em THRIFT, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.thrift** - Olá Mundo
2. **exemplo_2.thrift** - Variáveis e Tipos
3. **exemplo_3.thrift** - Controle de Fluxo
4. **exemplo_4.thrift** - Loops (Repetições)
5. **exemplo_5.thrift** - Funções
6. **exemplo_6.thrift** - Estruturas de Dados
7. **exemplo_7.thrift** - Tratamento de Erros
8. **exemplo_8.thrift** - Manipulação de Strings
9. **exemplo_9.thrift** - Matemática
10. **exemplo_10.thrift** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de THRIFT.
