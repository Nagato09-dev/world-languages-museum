3. **exemplo_3.cool** - Controle de Fluxo

5. **exemplo_5.cool** - Funções
6. **exemplo_6.cool** - Estruturas de Dados
7. **exemplo_7.cool** - Tratamento de Erros

9. **exemplo_9.cool** - Matemática
10. **exemplo_10.cool** - Conceito Avançado / Algoritmos
