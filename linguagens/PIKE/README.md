# PIKE

**Ano de Criação:** 1994

## Descrição

Esta pasta contém exemplos de código em PIKE, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.pike** - Olá Mundo
2. **exemplo_2.pike** - Variáveis e Tipos
3. **exemplo_3.pike** - Controle de Fluxo
4. **exemplo_4.pike** - Loops (Repetições)
5. **exemplo_5.pike** - Funções
6. **exemplo_6.pike** - Estruturas de Dados
7. **exemplo_7.pike** - Tratamento de Erros
8. **exemplo_8.pike** - Manipulação de Strings
9. **exemplo_9.pike** - Matemática
10. **exemplo_10.pike** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de PIKE.
Cada arquivo contém comentários explicativos em português.
