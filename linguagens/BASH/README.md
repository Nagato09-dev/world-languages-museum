# BASH

**Ano de Criação:** 1989

## Descrição

Esta pasta contém exemplos de código em BASH, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.bash** - Olá Mundo
2. **exemplo_2.bash** - Variáveis e Tipos
3. **exemplo_3.bash** - Controle de Fluxo
4. **exemplo_4.bash** - Loops (Repetições)
5. **exemplo_5.bash** - Funções
6. **exemplo_6.bash** - Estruturas de Dados
7. **exemplo_7.bash** - Tratamento de Erros
8. **exemplo_8.bash** - Manipulação de Strings
9. **exemplo_9.bash** - Matemática
10. **exemplo_10.bash** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de BASH.
