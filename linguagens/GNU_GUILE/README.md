# GNU_GUILE

**Ano de Criação:** 1993

## Descrição

Esta pasta contém exemplos de código em GNU_GUILE, organizados de forma progressiva para facilitar o aprendizado.

2. **exemplo_2.gnu_guile** - Variáveis e Tipos
3. **exemplo_3.gnu_guile** - Controle de Fluxo
4. **exemplo_4.gnu_guile** - Loops (Repetições)
5. **exemplo_5.gnu_guile** - Funções
6. **exemplo_6.gnu_guile** - Estruturas de Dados
7. **exemplo_7.gnu_guile** - Tratamento de Erros
8. **exemplo_8.gnu_guile** - Manipulação de Strings
9. **exemplo_9.gnu_guile** - Matemática
10. **exemplo_10.gnu_guile** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de GNU_GUILE.
