# TEX

**Ano de Criação:** 1978

## Descrição

Esta pasta contém exemplos de código em TEX, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.tex** - Olá Mundo
2. **exemplo_2.tex** - Variáveis e Tipos
3. **exemplo_3.tex** - Controle de Fluxo
4. **exemplo_4.tex** - Loops (Repetições)
5. **exemplo_5.tex** - Funções
6. **exemplo_6.tex** - Estruturas de Dados
7. **exemplo_7.tex** - Tratamento de Erros
8. **exemplo_8.tex** - Manipulação de Strings
9. **exemplo_9.tex** - Matemática
10. **exemplo_10.tex** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de TEX.
