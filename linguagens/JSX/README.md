# JSX

**Ano de Criação:** 2013

## Descrição

Esta pasta contém exemplos de código em JSX, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.jsx** - Olá Mundo
2. **exemplo_2.jsx** - Variáveis e Tipos
3. **exemplo_3.jsx** - Controle de Fluxo
4. **exemplo_4.jsx** - Loops (Repetições)
5. **exemplo_5.jsx** - Funções
6. **exemplo_6.jsx** - Estruturas de Dados
7. **exemplo_7.jsx** - Tratamento de Erros
8. **exemplo_8.jsx** - Manipulação de Strings
9. **exemplo_9.jsx** - Matemática
10. **exemplo_10.jsx** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de JSX.
