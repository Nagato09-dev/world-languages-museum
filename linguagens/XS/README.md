Esta pasta contém exemplos de código em XS, organizados de forma progressiva para facilitar o aprendizado.

2. **exemplo_2.xs** - Variáveis e Tipos
3. **exemplo_3.xs** - Controle de Fluxo
4. **exemplo_4.xs** - Loops (Repetições)
5. **exemplo_5.xs** - Funções
6. **exemplo_6.xs** - Estruturas de Dados
7. **exemplo_7.xs** - Tratamento de Erros
8. **exemplo_8.xs** - Manipulação de Strings
9. **exemplo_9.xs** - Matemática
10. **exemplo_10.xs** - Conceito Avançado / Algoritmos
