# BLADE

**Ano de Criação:** 2011

## Descrição

Esta pasta contém exemplos de código em BLADE, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.blade** - Olá Mundo
2. **exemplo_2.blade** - Variáveis e Tipos
3. **exemplo_3.blade** - Controle de Fluxo
4. **exemplo_4.blade** - Loops (Repetições)
5. **exemplo_5.blade** - Funções
6. **exemplo_6.blade** - Estruturas de Dados
7. **exemplo_7.blade** - Tratamento de Erros
8. **exemplo_8.blade** - Manipulação de Strings
9. **exemplo_9.blade** - Matemática
10. **exemplo_10.blade** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de BLADE.
