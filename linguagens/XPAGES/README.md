# XPAGES

**Ano de Criação:** 2007

## Descrição

Esta pasta contém exemplos de código em XPAGES, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.xpages** - Olá Mundo
2. **exemplo_2.xpages** - Variáveis e Tipos
3. **exemplo_3.xpages** - Controle de Fluxo
4. **exemplo_4.xpages** - Loops (Repetições)
5. **exemplo_5.xpages** - Funções
6. **exemplo_6.xpages** - Estruturas de Dados
7. **exemplo_7.xpages** - Tratamento de Erros
8. **exemplo_8.xpages** - Manipulação de Strings
9. **exemplo_9.xpages** - Matemática
10. **exemplo_10.xpages** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de XPAGES.
Cada arquivo contém comentários explicativos em português.
