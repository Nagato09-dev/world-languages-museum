# OPA

**Ano de Criação:** 2007

## Descrição

Esta pasta contém exemplos de código em OPA, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.opa** - Olá Mundo
2. **exemplo_2.opa** - Variáveis e Tipos
3. **exemplo_3.opa** - Controle de Fluxo
4. **exemplo_4.opa** - Loops (Repetições)
5. **exemplo_5.opa** - Funções
6. **exemplo_6.opa** - Estruturas de Dados
7. **exemplo_7.opa** - Tratamento de Erros
8. **exemplo_8.opa** - Manipulação de Strings
9. **exemplo_9.opa** - Matemática
10. **exemplo_10.opa** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de OPA.
