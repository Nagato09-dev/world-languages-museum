# GRASS

Esta pasta contém exemplos de código em GRASS, organizados de forma progressiva para facilitar o aprendizado.

2. **exemplo_2.grass** - Variáveis e Tipos
3. **exemplo_3.grass** - Controle de Fluxo
4. **exemplo_4.grass** - Loops (Repetições)
5. **exemplo_5.grass** - Funções
6. **exemplo_6.grass** - Estruturas de Dados
7. **exemplo_7.grass** - Tratamento de Erros
8. **exemplo_8.grass** - Manipulação de Strings
9. **exemplo_9.grass** - Matemática
10. **exemplo_10.grass** - Conceito Avançado / Algoritmos
