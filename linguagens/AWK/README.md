# AWK

**Ano de Criação:** 1977

## Descrição

Esta pasta contém exemplos de código em AWK, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.awk** - Olá Mundo
2. **exemplo_2.awk** - Variáveis e Tipos
3. **exemplo_3.awk** - Controle de Fluxo
4. **exemplo_4.awk** - Loops (Repetições)
5. **exemplo_5.awk** - Funções
6. **exemplo_6.awk** - Estruturas de Dados
7. **exemplo_7.awk** - Tratamento de Erros
8. **exemplo_8.awk** - Manipulação de Strings
9. **exemplo_9.awk** - Matemática
10. **exemplo_10.awk** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de AWK.
