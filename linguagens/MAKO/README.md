# MAKO

**Ano de Criação:** 2006

## Descrição

Esta pasta contém exemplos de código em MAKO, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.mako** - Olá Mundo
2. **exemplo_2.mako** - Variáveis e Tipos
3. **exemplo_3.mako** - Controle de Fluxo
4. **exemplo_4.mako** - Loops (Repetições)
5. **exemplo_5.mako** - Funções
6. **exemplo_6.mako** - Estruturas de Dados
7. **exemplo_7.mako** - Tratamento de Erros
8. **exemplo_8.mako** - Manipulação de Strings
9. **exemplo_9.mako** - Matemática
10. **exemplo_10.mako** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de MAKO.
