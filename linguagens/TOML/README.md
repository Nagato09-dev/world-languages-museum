# TOML

**Ano de Criação:** 2013

## Descrição

Esta pasta contém exemplos de código em TOML, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

2. **exemplo_2.toml** - Variáveis e Tipos
3. **exemplo_3.toml** - Controle de Fluxo
4. **exemplo_4.toml** - Loops (Repetições)
5. **exemplo_5.toml** - Funções
6. **exemplo_6.toml** - Estruturas de Dados
7. **exemplo_7.toml** - Tratamento de Erros
8. **exemplo_8.toml** - Manipulação de Strings
9. **exemplo_9.toml** - Matemática
10. **exemplo_10.toml** - Conceito Avançado / Algoritmos

## Como Usar
