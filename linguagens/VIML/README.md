# VIML

**Ano de Criação:** 1991

## Descrição

Esta pasta contém exemplos de código em VIML, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.viml** - Olá Mundo
2. **exemplo_2.viml** - Variáveis e Tipos
3. **exemplo_3.viml** - Controle de Fluxo
4. **exemplo_4.viml** - Loops (Repetições)
5. **exemplo_5.viml** - Funções
6. **exemplo_6.viml** - Estruturas de Dados
7. **exemplo_7.viml** - Tratamento de Erros
8. **exemplo_8.viml** - Manipulação de Strings
9. **exemplo_9.viml** - Matemática
10. **exemplo_10.viml** - Conceito Avançado / Algoritmos
