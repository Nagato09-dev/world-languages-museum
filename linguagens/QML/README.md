# QML

**Ano de Criação:** 2009

## Descrição

Esta pasta contém exemplos de código em QML, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.qml** - Olá Mundo
2. **exemplo_2.qml** - Variáveis e Tipos
3. **exemplo_3.qml** - Controle de Fluxo
4. **exemplo_4.qml** - Loops (Repetições)
5. **exemplo_5.qml** - Funções
6. **exemplo_6.qml** - Estruturas de Dados
7. **exemplo_7.qml** - Tratamento de Erros
8. **exemplo_8.qml** - Manipulação de Strings
9. **exemplo_9.qml** - Matemática
10. **exemplo_10.qml** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de QML.
