# PONY

**Ano de Criação:** 2015

## Descrição

Esta pasta contém exemplos de código em PONY, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.pony** - Olá Mundo
2. **exemplo_2.pony** - Variáveis e Tipos
3. **exemplo_3.pony** - Controle de Fluxo
4. **exemplo_4.pony** - Loops (Repetições)
5. **exemplo_5.pony** - Funções
6. **exemplo_6.pony** - Estruturas de Dados
7. **exemplo_7.pony** - Tratamento de Erros
8. **exemplo_8.pony** - Manipulação de Strings
9. **exemplo_9.pony** - Matemática
10. **exemplo_10.pony** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de PONY.
