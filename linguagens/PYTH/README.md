# PYTH

**Ano de Criação:** 2015

## Descrição

Esta pasta contém exemplos de código em PYTH, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.pyth** - Olá Mundo
2. **exemplo_2.pyth** - Variáveis e Tipos
3. **exemplo_3.pyth** - Controle de Fluxo
4. **exemplo_4.pyth** - Loops (Repetições)
5. **exemplo_5.pyth** - Funções
6. **exemplo_6.pyth** - Estruturas de Dados
7. **exemplo_7.pyth** - Tratamento de Erros
8. **exemplo_8.pyth** - Manipulação de Strings
9. **exemplo_9.pyth** - Matemática
10. **exemplo_10.pyth** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de PYTH.
