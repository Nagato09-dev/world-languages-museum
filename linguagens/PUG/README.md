# PUG

**Ano de Criação:** 2010

## Descrição

Esta pasta contém exemplos de código em PUG, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.pug** - Olá Mundo
2. **exemplo_2.pug** - Variáveis e Tipos
3. **exemplo_3.pug** - Controle de Fluxo
4. **exemplo_4.pug** - Loops (Repetições)
5. **exemplo_5.pug** - Funções
6. **exemplo_6.pug** - Estruturas de Dados
7. **exemplo_7.pug** - Tratamento de Erros
8. **exemplo_8.pug** - Manipulação de Strings
9. **exemplo_9.pug** - Matemática
10. **exemplo_10.pug** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de PUG.
Cada arquivo contém comentários explicativos em português.
