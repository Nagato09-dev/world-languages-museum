# ECL

**Ano de Criação:** 1999

## Descrição

Esta pasta contém exemplos de código em ECL, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.ecl** - Olá Mundo
2. **exemplo_2.ecl** - Variáveis e Tipos
3. **exemplo_3.ecl** - Controle de Fluxo
4. **exemplo_4.ecl** - Loops (Repetições)
5. **exemplo_5.ecl** - Funções
6. **exemplo_6.ecl** - Estruturas de Dados
7. **exemplo_7.ecl** - Tratamento de Erros
8. **exemplo_8.ecl** - Manipulação de Strings
9. **exemplo_9.ecl** - Matemática
10. **exemplo_10.ecl** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de ECL.
Cada arquivo contém comentários explicativos em português.
