# DART

**Ano de Criação:** 2011

## Descrição

Esta pasta contém exemplos de código em DART, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.dart** - Olá Mundo
2. **exemplo_2.dart** - Variáveis e Tipos
3. **exemplo_3.dart** - Controle de Fluxo
4. **exemplo_4.dart** - Loops (Repetições)
5. **exemplo_5.dart** - Funções
6. **exemplo_6.dart** - Estruturas de Dados
7. **exemplo_7.dart** - Tratamento de Erros
8. **exemplo_8.dart** - Manipulação de Strings
9. **exemplo_9.dart** - Matemática
10. **exemplo_10.dart** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de DART.
Cada arquivo contém comentários explicativos em português.
