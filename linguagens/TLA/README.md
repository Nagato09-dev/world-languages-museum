# TLA

**Ano de Criação:** 1999

## Descrição

Esta pasta contém exemplos de código em TLA, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.tla** - Olá Mundo
2. **exemplo_2.tla** - Variáveis e Tipos
3. **exemplo_3.tla** - Controle de Fluxo
4. **exemplo_4.tla** - Loops (Repetições)
5. **exemplo_5.tla** - Funções
6. **exemplo_6.tla** - Estruturas de Dados
7. **exemplo_7.tla** - Tratamento de Erros
8. **exemplo_8.tla** - Manipulação de Strings
9. **exemplo_9.tla** - Matemática
10. **exemplo_10.tla** - Conceito Avançado / Algoritmos

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de TLA.
