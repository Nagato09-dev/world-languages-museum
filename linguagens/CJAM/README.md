## Descrição

Esta pasta contém exemplos de código em CJAM, organizados de forma progressiva para facilitar o aprendizado.

2. **exemplo_2.cjam** - Variáveis e Tipos
3. **exemplo_3.cjam** - Controle de Fluxo
4. **exemplo_4.cjam** - Loops (Repetições)
5. **exemplo_5.cjam** - Funções
6. **exemplo_6.cjam** - Estruturas de Dados
7. **exemplo_7.cjam** - Tratamento de Erros
8. **exemplo_8.cjam** - Manipulação de Strings
9. **exemplo_9.cjam** - Matemática
10. **exemplo_10.cjam** - Conceito Avançado / Algoritmos
