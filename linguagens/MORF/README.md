## Descrição

Esta pasta contém exemplos de código em MORF, organizados de forma progressiva para facilitar o aprendizado.

2. **exemplo_2.morf** - Variáveis e Tipos
3. **exemplo_3.morf** - Controle de Fluxo
4. **exemplo_4.morf** - Loops (Repetições)
5. **exemplo_5.morf** - Funções
6. **exemplo_6.morf** - Estruturas de Dados
7. **exemplo_7.morf** - Tratamento de Erros
8. **exemplo_8.morf** - Manipulação de Strings
9. **exemplo_9.morf** - Matemática
10. **exemplo_10.morf** - Conceito Avançado / Algoritmos
