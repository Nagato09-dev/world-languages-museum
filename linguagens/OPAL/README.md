# OPAL

**Ano de Criação:** 1989

## Descrição

Esta pasta contém exemplos de código em OPAL, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.opal** - Olá Mundo
2. **exemplo_2.opal** - Variáveis e Tipos
3. **exemplo_3.opal** - Controle de Fluxo
4. **exemplo_4.opal** - Loops (Repetições)
5. **exemplo_5.opal** - Funções
6. **exemplo_6.opal** - Estruturas de Dados
7. **exemplo_7.opal** - Tratamento de Erros
8. **exemplo_8.opal** - Manipulação de Strings
9. **exemplo_9.opal** - Matemática
10. **exemplo_10.opal** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de OPAL.
