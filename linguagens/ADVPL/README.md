Esta pasta contém exemplos de código em ADVPL, organizados de forma progressiva para facilitar o aprendizado.

6. **exemplo_6.advpl** - Estruturas de Dados

10. **exemplo_10.advpl** - Conceito Avançado / Algoritmos
