# FISH

**Ano de Criação:** 2005

## Descrição

Esta pasta contém exemplos de código em FISH, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.fish** - Olá Mundo
2. **exemplo_2.fish** - Variáveis e Tipos
3. **exemplo_3.fish** - Controle de Fluxo
4. **exemplo_4.fish** - Loops (Repetições)
5. **exemplo_5.fish** - Funções
6. **exemplo_6.fish** - Estruturas de Dados
7. **exemplo_7.fish** - Tratamento de Erros
8. **exemplo_8.fish** - Manipulação de Strings
9. **exemplo_9.fish** - Matemática
10. **exemplo_10.fish** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de FISH.
