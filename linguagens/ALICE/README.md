# ALICE (Alice ML)

**Ano de Criação:** 1993

## Descrição

Alice ML é uma linguagem de programação funcional de propósito geral, de alto nível e multi-paradigma, baseada em Standard ML. Ela estende Standard ML com recursos como computação distribuída, programação de rede e threads. Os exemplos abaixo demonstram a sintaxe e os conceitos fundamentais de Alice ML.
