## Descrição

Esta pasta contém exemplos de código em APL, organizados de forma progressiva para facilitar o aprendizado.

2. **exemplo_2.apl** - Variáveis e Tipos

4. **exemplo_4.apl** - Loops (Repetições)
5. **exemplo_5.apl** - Funções
6. **exemplo_6.apl** - Estruturas de Dados
7. **exemplo_7.apl** - Tratamento de Erros
8. **exemplo_8.apl** - Manipulação de Strings
9. **exemplo_9.apl** - Matemática
10. **exemplo_10.apl** - Conceito Avançado / Algoritmos
