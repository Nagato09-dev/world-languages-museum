# ASP

**Ano de Criação:** 1996

## Descrição

Esta pasta contém exemplos de código em ASP, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.asp** - Olá Mundo
2. **exemplo_2.asp** - Variáveis e Tipos
3. **exemplo_3.asp** - Controle de Fluxo
4. **exemplo_4.asp** - Loops (Repetições)
5. **exemplo_5.asp** - Funções
6. **exemplo_6.asp** - Estruturas de Dados
7. **exemplo_7.asp** - Tratamento de Erros
8. **exemplo_8.asp** - Manipulação de Strings
9. **exemplo_9.asp** - Matemática
10. **exemplo_10.asp** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de ASP.
Cada arquivo contém comentários explicativos em português.
