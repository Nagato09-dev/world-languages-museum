# BRO

**Ano de Criação:** 2009

## Descrição

Esta pasta contém exemplos de código em BRO, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.bro** - Olá Mundo
2. **exemplo_2.bro** - Variáveis e Tipos
3. **exemplo_3.bro** - Controle de Fluxo
4. **exemplo_4.bro** - Loops (Repetições)
5. **exemplo_5.bro** - Funções
6. **exemplo_6.bro** - Estruturas de Dados
7. **exemplo_7.bro** - Tratamento de Erros
8. **exemplo_8.bro** - Manipulação de Strings
9. **exemplo_9.bro** - Matemática
10. **exemplo_10.bro** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de BRO.
