# SCHEME

**Ano de Criação:** 1975

## Descrição

Esta pasta contém exemplos de código em SCHEME, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

2. **exemplo_2.scheme** - Variáveis e Tipos
3. **exemplo_3.scheme** - Controle de Fluxo
4. **exemplo_4.scheme** - Loops (Repetições)
5. **exemplo_5.scheme** - Funções
6. **exemplo_6.scheme** - Estruturas de Dados
7. **exemplo_7.scheme** - Tratamento de Erros
8. **exemplo_8.scheme** - Manipulação de Strings
9. **exemplo_9.scheme** - Matemática
10. **exemplo_10.scheme** - Conceito Avançado / Algoritmos
