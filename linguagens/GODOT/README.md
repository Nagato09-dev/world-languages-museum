Esta pasta contém exemplos de código em GODOT, organizados de forma progressiva para facilitar o aprendizado.

6. **exemplo_6.godot** - Estruturas de Dados

9. **exemplo_9.godot** - Matemática
10. **exemplo_10.godot** - Conceito Avançado / Algoritmos
