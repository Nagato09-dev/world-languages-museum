## Descrição

Esta pasta contém exemplos de código em PUPPET, organizados de forma progressiva para facilitar o aprendizado.

2. **exemplo_2.puppet** - Variáveis e Tipos

4. **exemplo_4.puppet** - Loops (Repetições)
5. **exemplo_5.puppet** - Funções
6. **exemplo_6.puppet** - Estruturas de Dados
7. **exemplo_7.puppet** - Tratamento de Erros
8. **exemplo_8.puppet** - Manipulação de Strings
9. **exemplo_9.puppet** - Matemática
10. **exemplo_10.puppet** - Conceito Avançado / Algoritmos
