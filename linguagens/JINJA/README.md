Esta pasta contém exemplos de código em JINJA, organizados de forma progressiva para facilitar o aprendizado.

4. **exemplo_4.jinja** - Loops (Repetições)
5. **exemplo_5.jinja** - Funções
6. **exemplo_6.jinja** - Estruturas de Dados

9. **exemplo_9.jinja** - Matemática
10. **exemplo_10.jinja** - Conceito Avançado / Algoritmos
