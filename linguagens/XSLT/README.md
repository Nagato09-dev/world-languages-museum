# XSLT

**Ano de Criação:** 1999

## Descrição

Esta pasta contém exemplos de código em XSLT, organizados de forma progressiva para facilitar o aprendizado.

## Exemplos Inclusos

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.xslt** - Olá Mundo
2. **exemplo_2.xslt** - Variáveis e Tipos
3. **exemplo_3.xslt** - Controle de Fluxo
4. **exemplo_4.xslt** - Loops (Repetições)
5. **exemplo_5.xslt** - Funções
6. **exemplo_6.xslt** - Estruturas de Dados
7. **exemplo_7.xslt** - Tratamento de Erros
8. **exemplo_8.xslt** - Manipulação de Strings
9. **exemplo_9.xslt** - Matemática
10. **exemplo_10.xslt** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de XSLT.
Cada arquivo contém comentários explicativos em português.
