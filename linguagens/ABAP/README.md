# ABAP

**Ano de Criação:** 1983

## Descrição

Esta pasta contém exemplos de código em ABAP, organizados de forma progressiva para facilitar o aprendizado.

Os exemplos cobrem os seguintes tópicos:

1. **exemplo_1.abap** - Olá Mundo
2. **exemplo_2.abap** - Variáveis e Tipos
3. **exemplo_3.abap** - Controle de Fluxo
4. **exemplo_4.abap** - Loops (Repetições)
5. **exemplo_5.abap** - Funções
6. **exemplo_6.abap** - Estruturas de Dados
7. **exemplo_7.abap** - Tratamento de Erros
8. **exemplo_8.abap** - Manipulação de Strings
9. **exemplo_9.abap** - Matemática
10. **exemplo_10.abap** - Conceito Avançado / Algoritmos

## Como Usar

Navegue pelos exemplos em ordem para aprender os conceitos fundamentais de ABAP.
